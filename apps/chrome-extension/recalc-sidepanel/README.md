# ReCalc Chrome Extension

Fuente de la extensión Chrome MV3 para el panel lateral de ReCalc.

## Archivos clave

- `manifest.json`: manifiesto MV3
- `background.js`: configuración de `side_panel`
- `panel.html`: UI principal del panel lateral
- `panel.css`: estilos del panel
- `panel.js`: lógica de login, sync y cotización

## Publicación

El ZIP para Chrome Web Store debe generarse desde el contenido de esta carpeta, sin carpetas temporales ni archivos legacy del popup anterior.
